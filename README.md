# sokoban-solver-agent

An artificial intelligence project implementing an intelligent search 
agent capable of solving the Sokoban puzzle game.

## Overview
Sokoban is a classic combinatorial puzzle where a player pushes boxes 
to target locations in a warehouse. This project involved designing 
and implementing an AI agent using search algorithms to solve Sokoban 
puzzles of varying complexity, developed collaboratively in a team 
of three.

## Tools & Technologies
- Python — agent implementation and search algorithm development

## What I Did
- Designed and implemented an intelligent search agent using 
  heuristic and pathfinding algorithms to solve Sokoban puzzles
- Applied AI search techniques including informed and uninformed 
  search strategies
- Collaborated within a team of three to develop, test and validate 
  agent performance across multiple puzzle configurations
- Evaluated agent performance across varying levels of puzzle 
  complexity

## Key Skills Demonstrated
- Artificial intelligence and search algorithms
- Python programming
- Problem decomposition and algorithmic thinking
- Collaborative software development

## Results
Successfully developed an agent capable of solving Sokoban puzzles 
of varying difficulty, demonstrating effective application of AI 
search strategies.
